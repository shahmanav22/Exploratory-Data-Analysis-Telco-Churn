# Exploratory Data Analysis (EDA) - Telco Customer Churn

## Objective

Perform exploratory data analysis to understand customer behavior, identify churn patterns, and discover business insights using visualizations.

## Visualizations Included

- Count Plots
- Pie Chart
- Histograms
- Box Plots
- Violin Plot
- Scatter Plot
- Correlation Heatmap
- Pair Plot

## Key Insights

- Month-to-month contracts have the highest churn.
- Customers with higher monthly charges are more likely to churn.
- Long-tenure customers are more loyal.
- Fiber optic users exhibit higher churn rates.
- Customers lacking online security or tech support are more likely to leave.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
